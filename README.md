A fast and powerful Telegram Music Bot for playing songs in Voice Chats. Based on Pyrogram + PyTgCalls.

## Features
- Play music from YouTube
- Admin controls (pause, resume, stop, skip, etc.)
- Auth/Unauth system
- Broadcast messages
- Stats & user tracking
- Plugins: loop, seek, speed
